# 시작하며

1. SSL 인증서를 발급받고
2. 이를 나의 웹서버 (내 경우엔 AWS ELB를 사용할 예정이다.) 에 적용하는 방법을 알아본다.

# 나는 뭘 모르는가?

HTTPS 를 적용한다는것에 대해 기본지식도 있고 많이 알고 있다고 생각했다.
그런데 막상 하려니 정리가 되지 않는다.

- 인증서 적용을 ELB에 했다치자 클라이언트는 변경할것이 전혀 없는가?
- AWS에서 제공하는 free 공인 인증서를 만들고 있다. 도메인이 꼭 들어가야하는 이유는 뭔가?(ELB DNS명으로는 안되는가?)
- 만들어진 ELB DNS 명에 (A record) 라고 써있는 이유는 무엇인가
- 도메인 구입처(내 경우엔 Gabia 이다) 에서 생성된 ELB에 연결하고 싶다.
  A record 는 IP주소밖에 넣을수 없도록 되어있다. CNAME record 로
  api 에 ELB DNS 명을 넣어주었다. 왜 안되는가? (앞에 A record 라고 써있던것이 관계가 있을까?)
