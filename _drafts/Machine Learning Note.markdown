# Supervised Learning
----

Supervised Learning 은 우리가 이미 기대하는 Output 을 알고 있는 경우를 말한다.  
예를 들어보자.  
악성 종양 환자의 사진을 분석해서 이 사진의 환자가 양성인지 음성인지를 판단하는 머신 러닝 프로그램을 만든다고 가정한다.  
이 경우 우리가 기대하는 결과는 정해져있다. 양성이거나 음성이거나, 다른 경우의 수는 없다.  
이러한 경우를 Supervised Learning 이라 한다.  
Supervised Learning 은 다시 두가지로 분류할수 있다. Regressions and Classifications 이다.

## Regression
Regressions 는 우리가 기대하는 결과가 연속적인 경우를 말한다.  
우리가 부동산 시장에 나와있는 집들의 사이즈를가지고 있고 이를 바탕으로 이 집의 금액을 예측하는 머신러닝  
프로그램을 만든다고 해보자. 집들의 금액은 어떠할까 1억 일수도 있고, 1억 2천일수도 있고, 100억 일수도 있다.  
우리가 기대하는 결과는 연속적인 경우의 수중 하나의 값이 될것이다. 이러한 경우가 Regressions Problem 이다.
