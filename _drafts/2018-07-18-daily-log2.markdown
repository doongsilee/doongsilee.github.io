# 공격적 대화의 유형

묘하게 웃으면서 말하는데 듣는사람이 화가 나고 대화가 이어지기 힘든 대화법이 있다.
근데 이게 또 묘하게 다른사람에게 설명하려고 시도해보면 걍 그렇다.
듣는 사람 입장에선 그게 왜 화나지? 하게 되는 느낌.

왜그럴까? 한번 적어보자.
앞으로 용의자(?)를 X 로 그 외 대화하는 사람을 다른 알파벳으로 표시한다.

X : 인터넷이 또 끊어졌어
A : 그러게요 이거 돈내고 설치한 인터넷인데도 끊어지네요
X : 당연히 인터넷이 건물 전체가 끊어지면 다 끊어지지 뭐 우리만 돈낸다고 다르나?

B : X 님 우리 서버에 이런 기능 있나요?
X : 그런거 없지요. 이야 그런거 고객 한 천명 있으면 개발해야될것 같은 기능인데? 항공모함 이거 언제 띄울꺼야?(웃으며)
