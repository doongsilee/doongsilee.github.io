# Weekly review

![Image]({{ "/assets/181022.png" | relative_url }}){: width="50%"}


100일 플랜 시작의 첫주였기에
좀 널널하게 계획을 잡았다고 생각했는데 역시나 계획을 완수하는건 어렵다.
원인이 뭘까? 절대적인 시간이 부족한걸까?
그것도 원인중에 하나일 수는 있을것이다.

그래서 오늘은 아침에 일찍 일어나보았다.
아침에 일찍 일어나서 운동과 블로그를 오전중에 처리하면
영어와 블록체인을 저녁에 할 시간이 생길것이라고 생각했는데..
저녁에 퇴근하고 오면 게임을 하고 싶은 마음 / 와이프와 맥주 한잔 하고 싶은 마음과도 싸워야 한다는걸 깜빡했다.
(이게 정말 무시무시하다)

일단 의무적으로 하는 게임은 좀 치워야겠다. 엔딩은 좀 아쉽지만 다음 기회에 보자.
다음주 계획도 이번주와 비슷하게 조금은 널널하다 싶게 잡고 시작하자.
