# 시작하며

신규 앱을 런칭했다.
플레이스토어(혹은 앱스토어)에 런칭했고, 페이스북 광고와
플레이스토어 검색어 광고를 걸어서 이제 어느정도 고객들이 유입되기 시작했다.

이제부터 뭘할까? 보통은 서비스를 개선하고자 할것이다.
그런데 뭐부터 개선하지?? 뭐..우리 마음대로 하려면 못할것도 없겠지만
아무래도 가장 고객들이 많이 원하는 부분부터 개선하는게 가장 효과적이지 않을까.

그렇기에 모든 서비스 운영과 마케팅의 시작은
고객들이 우리앱을(서비스를) 맘에 들어하는지, 만약 맘에 들지 않는다면
어디가 특히 그런것인지 측정하는것에서부터 이야기가 시작된다.
이 분야에서 유명한 책은 그 유명한 라이언 홀리데이의 저서 '그로스 해킹'이다.
보다 자세한 내용은 책을 읽어보면 좋을 것 같다.

왜 해야하는지는 대충 이해됫으니 우리는 조금더 실무적인 관점에서
당장 어떻게 할수 있는지 접근해보도록 한다.
